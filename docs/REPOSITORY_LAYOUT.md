# Repository layout

```text
.github/                 GitHub templates and CI
schemas/                 Public report schemas
src/nzp_t4_inspector/    Current read-only inspector
 tests/                   Unit tests and original fixtures
 tools/                   Local validation scripts
 upstream/                Upstream coordination metadata
 docs/                    Public verified documentation
```

Engine, gameplay, Android, and compatibility-runtime source directories are intentionally absent until their source relationship and build path are established.
