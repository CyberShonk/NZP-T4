# Contributing

NZ:P T4 is in a foundation stage. Contributions should be small, testable, and compatible with the repository content policy.

## Before opening a change

- Read `PROJECT_PROCESS.md`.
- Run `./tools/check-project.sh`.
- Do not attach or commit game files, map packages, extracted assets, copied scripts, or native framework binaries.
- Use independently authored test data.
- Describe observed behavior without pasting proprietary content.

## Compatibility reports

A report may be shared only after checking that it contains no personal paths, package payload, copied script text, or recoverable asset data. Hashes, counts, classifications, stable error codes, and sanitized logs are preferred.

## Licensing

By contributing original code, you agree that it may be distributed under GPL-2.0-only. Do not submit code copied from a source with incompatible or uncertain licensing.
