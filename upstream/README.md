# Upstream coordination

This directory records public metadata for upstream components. It does not vendor or clone upstream source.

1. Create explicit project forks of `nzp-team/fteqw` and `nzp-team/quakec`.
2. Record exact base SHAs and dates.
3. Preserve GPL notices and repository history.
4. Document remotes and update policy.
5. Prove the unmodified baseline before compatibility changes.
